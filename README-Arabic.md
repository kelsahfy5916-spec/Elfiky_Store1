# 👑 دليل تشغيل ونشر موقع Elfiky_Store (بالعربي)

أهلاً بك يا هندسة في مشروع موقع **Elfiky_Store** المتطور جداً! 

الكثير من المطورين المعتادين على بيئة (PHP / MySQL / XAMPP) يسألون: **"هل لو حطيت الملفات في مجلد `htdocs` وشغلت `xampp` الموقع هيشتغل؟"**
والإجابة المباشرة هي: **لا، لن يعمل داخل XAMPP.**

---

## ❓ ليه مش هيشتغل على XAMPP؟
برنامج **XAMPP** مخصص لتشغيل المواقع المبنية بلغة **PHP** وقواعد بيانات **MySQL** (عبر خادم Apache التقليدي).

بينما موقعك الفاخر مبني بأحدث التقنيات البرمجية العالمية لعام 2026:
1. **Next.js 16 و React 19**: أحدث إطار عمل في العالم لبناء واجهات مستخدم فائقة السرعة وتفاعلية.
2. **Node.js**: بيئة التشغيل الحديثة (JavaScript Runtime) التي تعمل كخادم بديل لـ Apache.
3. **PostgreSQL**: نظام قواعد البيانات الاحترافي الأقوى والأكثر أماناً في العالم (بديل MySQL)، ومتصل بالموقع عبر مكتبة **Drizzle ORM**.

---

## 💻 إزاي تشغله على جهازك الشخصي (Local Computer) بالطريقة الصحيحة 100%؟

لتشغيل الموقع على جهازك (ويندوز، ماك، أو لينكس) اتبع الـ 4 خطوات البسيطة دي:

### 1. تثبيت البرامج الأساسية:
- **تثبيت Node.js**: نزل وثبت أحدث إصدار مستقر (LTS) من الرابط الرسمي: [Node.js Official](https://nodejs.org)
- **تثبيت PostgreSQL**: نزل وثبت قاعدة البيانات من الرابط الرسمي: [PostgreSQL Official](https://www.postgresql.org/download/)
  *(⚠️ ملاحظة هامة: أثناء تثبيت PostgreSQL سيطلب منك كتابة كلمة مرور (Password) للمستخدم الأساسي `postgres`، احفظ الكلمة دي كويس جداً لأنك هتحتاجها).*

### 2. تثبيت مكتبات المشروع:
- بعد ما تفك ضغط ملفات الموقع، افتح سطر الأوامر (`Terminal` أو `CMD` أو `Git Bash`) داخل مجلد المشروع.
- اكتب الأمر ده وادوس Enter لتثبيت كل المكتبات (React, Framer Motion, Drizzle):
  ```bash
  npm install
  ```

### 3. ربط قاعدة البيانات:
- أنشئ ملف جديد في مجلد المشروع الرئيسي وسميه **`.env`** (تأكد من النقطة في الأول).
- افتح الملف واكتب فيه سطر الاتصال بقاعدة البيانات بتاعتك بالشكل ده:
  ```env
  DATABASE_URL="postgresql://postgres:YOUR_PASSWORD@localhost:5432/elfiky_db"
  ```
  *(استبدل `YOUR_PASSWORD` بكلمة المرور اللي كتبتها وانت بتثبت PostgreSQL في الخطوة 1).*

### 4. بناء الجداول وتشغيل الموقع:
- افتح الـ Terminal واكتب الأمر ده عشان تبني الجداول في قاعدة البيانات وتضيف المنتجات واليوزرز الأولية:
  ```bash
  npx drizzle-kit push
  ```
- أخيراً، عشان تشغل الموقع في وضع التطوير والتعديل (Development Mode) اكتب:
  ```bash
  npm run dev
  ```
- افتح متصفحك على الرابط `http://localhost:3000` ومبروك عليك الموقع شغال زي الفل! 🚀

## 🌐 نصيحة ذهبية: إزاي ترفعه للجمهور مجاناً (Cloud Production Deployment)

بدل ما تشغله على جهازك بس، تقدر ترفعه مجاناً وبكل سهولة ليكون متاح للزبائن الحقيقية 24 ساعة:
1. ارفع ملفات المشروع بالكامل على حسابك في **GitHub**.
2. أنشئ قاعدة بيانات PostgreSQL سحابية مجانية بضغطة زر على منصة **[Supabase](https://supabase.com)** أو **[Neon.tech](https://neon.tech)**، وانسخ رابط الـ `DATABASE_URL` بتاعها.
3. ادخل على منصة الاستضافة المجانية الرائعة **[Vercel](https://vercel.com)** (الشركة المطورة لـ Next.js)، واعمل استيراد (`Import`) لمشروعك من GitHub.
4. في خانة الـ `Environment Variables` داخل Vercel، حط متغير باسم `DATABASE_URL` وحط قدامه رابط قاعدة البيانات السحابية اللي نسخته.
5. اضغط **Deploy** ومبروك عليك رابط احترافي مباشر لموقعك يعمل مدى الحياة! 💎✨
